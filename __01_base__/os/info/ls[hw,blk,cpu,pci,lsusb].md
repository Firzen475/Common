## Примеры использования lspci

| Команда | Описание | Пример вывода |
|---------|----------|---------------|
| **Базовые команды** |
| `lspci` | Показать список всех PCI-устройств | `00:00.0 Host bridge: Intel Corporation 8th Gen Core Processor Host Bridge/DRAM Registers (rev 07)` |
| `lspci -v` | Подробная информация о устройствах | Добавляет сведения о IRQ, памяти, драйверах |
| `lspci -vv` | Максимально подробный вывод | Показывает все возможные детали |
| `lspci -k` | Показать используемые драйверы | `Kernel driver in use: i915` |
| **Фильтрация устройств** |
| `lspci -s 00:02.0` | Информация о конкретном устройстве | `00:02.0 VGA compatible controller: Intel Corporation UHD Graphics 630 (Desktop)` |
| `lspci -d 8086:3e92` | Найти устройство по ID вендора:устройства | `00:02.0 VGA compatible controller: Intel Corporation...` |
| `lspci -d ::0300` | Показать только видеокарты | Список всех GPU |
| `lspci -d ::0200` | Показать только сетевые карты | Список Ethernet/Wi-Fi адаптеров |
| **Специальные форматы вывода** |
| `lspci -t` | Древовидное представление | ```-+-[0000:00]-+-00.0`<br> `  +-02.0`<br> `  +-14.0``` |
| `lspci -nn` | Показать ID устройств | `00:14.0 USB controller [0c03]: Intel Corporation 200 Series/Z370... [8086:a2af]` |
| `lspci -mm` | Машиночитаемый формат | `Slot: 00:1f.4`<br>`Class: Serial bus controller`<br>`Vendor: Intel Corporation` |
| **Дополнительные возможности** |
| `lspci -vv \| grep -i pcie` | Проверить версию PCI Express | `LnkCap: Port #0, Speed 8GT/s, Width x16` |
| `lspci -xxxx` | Показать конфигурационное пространство PCI | Шестнадцатеричный дамп регистров |
| `lspci -b` | Показать устройства с точки зрения шины | Альтернативное представление адресов |

## Примеры использования lsusb

| Команда | Описание | Пример вывода |
|---------|----------|---------------|
| **Базовые команды** |
| `lsusb` | Список всех USB-устройств | `Bus 001 Device 002: ID 8087:0024 Intel Corp.` |
| `lsusb -v` | Подробная информация | Детальные тех. характеристики |
| `lsusb -t` | Древовидное представление | `/: Bus 01.Port 1: Dev 1, Class=root_hub...` |
| **Фильтрация устройств** |
| `lsusb -d 0781:5581` | Поиск по ID вендора:устройства | `Bus 001 Device 003: ID 0781:5581 SanDisk Corp.` |
| `lsusb -s 001:002` | Поиск по шине и устройству | `Bus 001 Device 002: ID 8087:0024 Intel Corp.` |
| **Специальные форматы** |
| `lsusb -vv` | Максимально подробный вывод | Все технические параметры |
| `lsusb --tree` | Альтернативное древовидное представление | Аналогично `-t` но с отступами |
| **Полезные комбинации** |
| `lsusb \| grep -i keyboard` | Поиск клавиатур | `Bus 003 Device 002: ID 046d:c31c Logitech...` |
| `lsusb \| grep -i storage` | Поиск накопителей | `Bus 002 Device 003: ID 0781:5581 SanDisk...` |
| `lsusb -v \| grep -Ei "(MaxPower\|bcdUSB)"` | Проверка версии USB и питания | `bcdUSB 2.00`, `MaxPower 500mA` |
| **Расширенная диагностика** |
| `lsusb -D /dev/bus/usb/001/002` | Информация из дескриптора устройства | RAW-данные конкретного устройства |
| `lsusb -v 2>/dev/null \| grep -A3 bInterfaceClass.*HID` | Поиск HID-устройств | Вывод только устройств ввода |
