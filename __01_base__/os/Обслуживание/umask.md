## umask (user file creation mask)
* это маска прав доступа, которая определяет, какие права будут убраны у вновь создаваемых файлов и директорий.
* Формула
  * Файл `666-umask -> 666-0022=644`
  * Директория `777-umask -> 777-0022=755`
* Посмотреть `umask`
* Установить `umask 0027` Постоянно - добавить в `~/.bashrc` или `/etc/profile`

## Специальные биты
* первая цифра в маске прав `chmod 4755`
* `setuid (SUID, 4)` — запуск файла с правами владельца.
* `setgid (SGID, 2)` — запуск файла с правами группы или наследование группы для директорий.
* `sticky bit (1)` — защита файлов в общей директории от удаления другими пользователями
