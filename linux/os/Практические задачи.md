
## Список подключенных к серверу пользователей
```shell
# Текущие пользователи
who
# История входов
last
# Сессии sshd 
ps aux | grep sshd
# 


```

## Как запретить пользователю логиниться по SSH, но разрешить SFTP?
```yaml
Match User username
    ForceCommand internal-sftp
    ChrootDirectory /home/username
    PermitTunnel no
    X11Forwarding no
    AllowTcpForwarding no
```
