# Команда dmidecode: справочное руководство

## Основные команды

| Ключ               | Описание                                                                 | Пример использования                     |
|--------------------|-------------------------------------------------------------------------|------------------------------------------|
| `-t`, `--type`     | Вывод информации по указанному DMI-типу                                 | `sudo dmidecode -t 1`                    |
| `-s`, `--string`   | Вывод конкретного значения                                              | `sudo dmidecode -s bios-version`         |
| `-H`, `--handle`   | Вывод данных по конкретному DMI-дескриптору                             | `sudo dmidecode -H 0x002A`               |
| `-u`, `--dump`     | Вывод необработанных данных DMI                                         | `sudo dmidecode -u`                      |
| `--no-quirks`      | Отключение автоматических исправлений                                   | `sudo dmidecode --no-quirks`             |
| `-q`, `--quiet`    | Тихий режим                                                             | `sudo dmidecode -q -t 4`                 |
| `-h`, `--help`     | Вывод справки                                                           | `dmidecode -h`                           |
| `-V`, `--version`  | Вывод версии утилиты                                                    | `dmidecode -V`                           |

## Примеры использования

### Системная информация

| Команда                          | Вывод                                                                 |
|----------------------------------|----------------------------------------------------------------------|
| `sudo dmidecode -t 0`            | Информация о BIOS (версия, дата, производитель)                     |
| `sudo dmidecode -t 1`            | Данные о системе (производитель, модель, серийный номер)             |
| `sudo dmidecode -t 2`            | Информация о материнской плате                                      |
| `sudo dmidecode -t 3`            | Данные о корпусе устройства                                         |

### Процессор и память

| Команда                          | Вывод                                                                 |
|----------------------------------|----------------------------------------------------------------------|
| `sudo dmidecode -t 4`            | Характеристики процессора                                           |
| `sudo dmidecode -t 5`            | Информация о контроллере памяти                                     |
| `sudo dmidecode -t 6`            | Данные о модулях памяти                                             |
| `sudo dmidecode -t 16`           | Физическая организация памяти                                       |
| `sudo dmidecode -t 17`           | Детализация модулей RAM (DIMM)                                      |

### Полезные сокращения

| Команда                          | Вывод                                                                 |
|----------------------------------|----------------------------------------------------------------------|
| `sudo dmidecode -s system-uuid`  | UUID системы                                                        |
| `sudo dmidecode -s system-serial-number` | Серийный номер системы                                      |
| `sudo dmidecode -s baseboard-product-name` | Модель материнской платы                               |
