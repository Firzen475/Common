
## Использование

```shell
# Все сигналы
kill -l

# Убить процесс
pkill -9 firefox || kill $(pgrep firefox)

# Убить дерево
pkill -P <PPID>  # убьёт все дочерние процессы
kill <PPID> 

# Перезапустить демон
kill -1 $(pgrep nginx)

kill -19 1234    # остановить (SIGSTOP)
kill -18 1234    # продолжить (SIGCONT)
```


## Основные сигналы
| Сигнал    | Код | Важность | Описание                                                                                                         |
|:----------|:---:|:--------:|:-----------------------------------------------------------------------------------------------------------------|
| SIGHUP    |  1  |    2     | Потеря соединения с управляющим терминалом (прерывание связанных программ); <br/> Для демонов переинициализация; |
| SIGINT    |  2  |    1     | Прерывание из терминала (Ctrl + C).                                                                              |
| SIGKILL   |  9  |    1     | Безусловное завершение процесса. (кроме зомби и init).                                                           |
| SIGTERM   | 15  |    1     | Запрос на завершение программы (kill посылает по умолчанию).                                                     |
| SIGCONT   | 18  |    2     | Посылается для возобновления процесса после SIGTSTP, SIGTTIN, SIGTTOU.                                           |
| SIGSTOP   | 19  |    2     | Безусловная пауза процесса.                                                                                      |
