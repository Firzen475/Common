

| Ключ    | Описание                                | Пример      |
|---------|-----------------------------------------|-------------|
|         | `who`                                   |
| `-a`    | Показать все сессии (включая системные) | `who -a`    |
| `-b`    | Время последней загрузки системы        | `who -b`    |
| `-d`    | Показать "мертвые" процессы             | `who -d`    |
| `-H`    | Вывести заголовки столбцов              | `who -H`    |
| `-i`    | Показать время простоя (idle)           | `who -i`    |
| `-m`    | Только текущая сессия пользователя      | `who -m`    |
| `-q`    | Быстрый подсчет пользователей           | `who -q`    |
| `-r`    | Текущий уровень выполнения (runlevel)   | `who -r`    |
| `-u`    | Детальная информация о сессиях          | `who -u`    |
| `--ips` | Показывать IP-адреса вместо имен хостов | `who --ips` |
|         | `w`                                     |
| -h      | 	Скрыть заголовок	w -h                  |
| -i      | 	Показывать IP-адреса	w -i              |
| -o      | 	Старый формат вывода	w -o              |
| -s      | 	Короткий формат (без JCPU/PCPU)	w -s   |
| -u      | 	Игнорировать имена пользователей	w -u  |
| -f      | 	Показывать/скрыть поле FROM	w -f       |

**Пример вывода:**

w -i
